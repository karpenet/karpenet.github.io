First Post

Sun 15 May 2016

This is my first post on the blog. This blog will be about my endeavors in electronics, spanning from embedded systems to swarm robotics. At present I would be posting my project logs on this website, in future I plan to create tutorials on this page. The jargon used has been made noob friendly, if you still have any questions, feel free to contact me.

For most of my microcontroller/microprocessor based projects, I intend use the AVR (of course from Atmel!) and ARM (mostly from ST Microelectronics). All the softwares used are Mac OS friendly, you may also use Windows alternatives.

All the projects are Open-Source and the relevant files can be found on Github. This blog is created using Pelican, which is a Python based framework.

Happy Making!
